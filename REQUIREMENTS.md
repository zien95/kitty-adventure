# Kitty Adventure Minimum System Requirements

Version: v26.8.2

## Android
- Android 7.0 or newer, API 24+
- 2 GB RAM
- About 150 MB free storage

## Mobile sideload build
- iOS 15.0 or newer
- 2 GB RAM
- About 150 MB free storage
- A sideloading tool is needed to install the unsigned IPA build

## macOS
- macOS 12 Monterey or newer
- Intel or ARM-based Mac
- 4 GB RAM
- About 200 MB free storage

## Web
- Latest Chrome, Edge, Safari, or Firefox
- JavaScript enabled
- WebGL/canvas support
- 2 GB RAM

## Controls
- Touch, mouse, or keyboard
- Racing supports tapping/clicking or pressing Space repeatedly
