# Pet Game Image Assets

## Required Images

Replace the placeholder files with actual PNG images:

### Main Images
- **cat_sitting.png** (200x200px) - Cartoonish sitting cat character
- **kitty_pet.png** - Front-facing mood base used by the game
- **kitty_pet_happy.png** - Happy/default portrait crop
- **kitty_pet_excited.png** - Excited/playful portrait crop
- **kitty_pet_sleepy.png** - Sleepy/sad/sick portrait crop
- **kitty_pet_loving.png** - Loving portrait crop
- **background_outdoor.png** (1080x1920px) - Outdoor scene with green environment and house

### Icon Images (48x48px)
- **coin_icon.png** - Gold coin, circular, shiny
- **heart_icon.png** - Red heart, cartoonish, filled
- **food_icon.png** - Meat/fish/pet food, colorful
- **toy_icon.png** - Ball/toy mouse, colorful
- **clean_icon.png** - Soap/sponge/bubbles, blue/white
- **shop_icon.png** - Shopping cart/store front, colorful

## Where to Get Images

### Free Options:
1. **Freepik** - Search for "cartoon cat", "pet icons", "game UI"
2. **Flaticon** - Free icon sets for games
3. **OpenGameArt.org** - Free game assets
4. **Itch.io** - Free asset packs

### Paid Options:
1. **Envato Elements** - Professional game assets
2. **Adobe Stock** - High-quality illustrations
3. **Shutterstock** - Large image library

## Style Guidelines
- Cartoonish, cute style
- Bright, vibrant colors
- Consistent art style across all images
- Transparent backgrounds (PNG format)
- Mobile-friendly resolution

## Installation
1. Download or create images
2. Replace placeholder files in this directory
3. Keep the same filenames
4. Run `flutter pub get` if needed
5. Restart the app
