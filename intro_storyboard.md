## 🎨 Animation Notes

### Transitions
- **Cut:** Quick scene changes (montage)
- **Fade:** Smooth transitions (UI reveal)
- **Zoom:** Dramatic emphasis (hook shot)
- **Slide:** Text appearances

### Effects
- **Sparkles:** ✨ Pet interactions, rewards
- **Hearts:** ❤️ Pet happiness, affection
- **Coins:** 🪙 Currency, rewards
- **Stars:** ⭐ Level ups, achievements
- **Bounce:** Pet reactions, button presses

### Timing Guide
- **Fast cuts:** 0.5-1 second (montage)
- **Medium pace:** 2-3 seconds (features)
- **Slow reveal:** 3-5 seconds (intro/outro)

## 🎵 Audio Cues

| Time | Visual | Audio | Purpose |
|------|--------|-------|---------|
| 0s | Pet reveal | Chime + meow | Hook attention |
| 3s | UI reveal | Music starts | Set tone |
| 8s | Pet interaction | Happy sound | Show engagement |
| 12s | Buttons | Click sounds | Show interactivity |
| 16s | Progress | Coin sounds | Show rewards |
| 20s | Montage | Music builds | Excitement |
| 30s | CTA | Music peak | Call to action |
| 35s | Logo | Music fade | Branding |

## 📱 Technical Specs

### Export Formats
- **TikTok/Reels:** 1080x1920, 30fps, MP4
- **YouTube Shorts:** 1080x1920, 30fps, MP4  
- **App Store:** 1080x1920 or 1920x1080, MP4
- **Website:** 1920x1080, MP4 (for desktop)

### File Sizes
